# SocioSentiment – Multilingual Sentiment & Emotion Analysis

SocioSentiment is a machine learning and NLP based web application that analyzes public opinion on social issues using multilingual text inputs. The system performs both sentiment classification and emotion detection to provide deeper insights into public reactions.

## Features
- Multilingual text analysis (English + Hindi)
- Sentiment classification: Positive, Negative, Neutral
- Emotion detection: Happy, Sad, Angry, Fear, Surprise
- Confidence score for predictions
- Explainable AI with keyword highlighting
- Interactive web interface using Streamlit

## Technologies Used
- Python
- Natural Language Processing (NLP)
- Scikit-learn
- NLTK
- Pandas & NumPy
- Streamlit
- TF-IDF / Word Embeddings

## System Workflow
1. User inputs text through the web interface
2. Text preprocessing (cleaning, tokenization, stopword removal)
3. Feature extraction using NLP techniques
4. Sentiment and emotion classification using ML models
5. Confidence score and explanation generation
6. Results displayed through interactive visualizations

## Applications
- Public opinion monitoring
- Social media sentiment analysis
- Policy feedback analysis
- NGO campaign evaluation
- Market and social research

## Future Improvements
